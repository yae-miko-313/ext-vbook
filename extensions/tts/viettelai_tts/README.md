# Viettel AI TTS Extension

Tích hợp **Viettel AI Text-to-Speech** vào vBook. 16 giọng đọc tiếng Việt Bắc/Trung/Nam.

## Lấy Token

1. Truy cập [viettelai.vn](https://viettelai.vn) và đăng nhập
2. Vào **Dashboard → Token**
3. Copy token và dán vào phần **Cài đặt Extension** trong vBook

## Giới hạn

- Miễn phí **50.000 ký tự** tháng đầu
- Gia hạn **20.000 / 50.000 ký tự** mỗi tháng
- Tối đa **5.000 ký tự/request**
