# VBook Extension Catalog Viewer

Quick visual dashboard để xem tổng quan các extensions có sẵn.

## How to Use

### Option 1: Open in Browser (Simplest)
```bash
# Windows: Right-click index.html → Open with → Browser
# Or double-click index.html
```

### Option 2: Use Python Server
```bash
# Python 3.x
cd web
python -m http.server 8000

# Then open: http://localhost:8000
```

### Option 3: Use Node.js
```bash
# Install http-server globally
npm install -g http-server

# Run
cd web
http-server

# Then open: http://localhost:8080
```

## Features

✨ **Dashboard Overview**
- Tổng số extensions: 372+
- Stats theo type (Novel, Comic, Chinese Novel, etc.)

🔍 **Search & Filter**
- Tìm kiếm theo tên, author, description
- Filter theo type
- Sort: A-Z, Version (mới nhất), Author

📊 **Visual Cards**
- Extension name, version, author
- Description preview
- Type badge (Novel, Comic, etc.)
- Link to source website
- Relative path to folder

## Data Source

Loads từ `../extensions/plugin.json` (auto-generated by `npx vbook build-catalog`)

### Important Notes

⚠️ **CORS Issues:**
- Khi open file:// trực tiếp, có thể gặp CORS error
- **Giải pháp:** Dùng Python/Node.js server hoặc browse qua localhost

🔄 **Auto Update:**
- Mỗi khi chạy `npx vbook build-catalog`, file `extensions/plugin.json` được update
- Refresh browser để thấy dữ liệu mới

## Troubleshooting

**Q: Không hiển thị extensions?**
A: 
1. Chắc chẩn đã chạy `npx vbook build-catalog` trước
2. Kiểm tra browser console (F12) xem error gì
3. Sử dụng server (Python/Node) thay vì open file://

**Q: Có thể edit/export data?**
A: Hiện tại chỉ view. Edit trực tiếp extensions/ folders hoặc dùng CLI commands.
